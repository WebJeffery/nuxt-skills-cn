# 点击标记

根据点击进度高亮和自动滚动演讲者备注。

## 语法

在演讲者备注中添加 `[click]` 标记：

```md
<!--
第一次点击之前的内容

[click] 这将在第一次点击后高亮显示

第一次点击后也高亮显示

- [click] 此列表元素在第二次点击后高亮显示

[click:3] 最后一次点击（跳过两次点击）
-->
```

## 行为

- 标记之间的备注与幻灯片进度同步高亮
- 自动滚动演讲者视图到活动部分
- 使用 `[click:{n}]` 跳转到特定点击次数

## 要求

- 仅在演讲者模式下工作
- 备注必须是幻灯片末尾的 HTML 注释
