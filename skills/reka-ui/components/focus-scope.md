# 焦点范围

焦点陷阱容器

**部分：** `FocusScope`

## FocusScope

### 属性

| 属性      | 类型                 | 默认值 |
| --------- | -------------------- | ------- |
| `as`      | `AsTag \| Component` | `"div"` |
| `asChild` | `boolean`            | -       |
| `loop`    | `boolean`            | `false` |
| `trapped` | `boolean`            | `false` |

### 事件

| 事件              | 载荷          |
| ------------------ | ---------------- |
| `mountAutoFocus`   | `[event: Event]` |
| `unmountAutoFocus` | `[event: Event]` |
