# 聊天消息列表

显示聊天消息列表，旨在与 Vercel AI SDK 无缝协作。

## 关键属性

- `messages`: 显示聊天消息列表。
- `status`: 当助手正在处理时显示视觉指示器。
- `user`: 更改 `user` 消息的 [ChatMessage](/docs/components/chat-message) 属性。
- `assistant`: 更改 `assistant` 消息的 [ChatMessage](/docs/components/chat-message) 属性。

## 用法

```vue
<UChatMessages
  <!-- props here -->
/>
```
