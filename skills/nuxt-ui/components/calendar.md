# 日历

用于选择单个日期、多个日期或日期范围的日历组件。

> 基于 [Reka UI Calendar](https://reka-ui.com/docs/components/calendar)

## 关键属性

- `multiple`: 允许多选。
- `range`: 选择日期范围。
- `color`: 更改日历的颜色。
- `variant`: 更改日历的变体。
- `size`: 更改日历的大小。
- `disabled`: 禁用日历。
- `numberOfMonths`: 更改日历中的月份数。
- `weekNumbers`: 显示周数（v4.4+）。

## 用法

```vue
<UCalendar
  <!-- props here -->
/>
```
