# 复选框

在选中状态和未选中状态之间切换的输入元素。

> 基于 [Reka UI Checkbox](https://reka-ui.com/docs/components/checkbox)

## 关键属性

- `label`: 设置复选框的标签。
- `description`: 设置复选框的描述。
- `icon`: 设置复选框选中时的图标。
- `color`: 更改复选框的颜色。
- `variant`: 更改复选框的变体。
- `size`: 更改复选框的大小。
- `indicator`: 更改位置或隐藏指示器。
- `disabled`: 禁用复选框。

## 用法

```vue
<UCheckbox
  <!-- props here -->
/>
```
