# 文件上传

用于上传文件的输入元素。

## 关键属性

- `multiple`: 允许多选文件。
- `dropzone`: 启用/禁用可拖放区域。
- `interactive`: 启用/禁用可点击区域。
- `accept`: 指定输入的允许文件类型。
- `label`: 设置文件上传的标签。
- `description`: 设置文件上传的描述。
- `icon`: 设置文件上传的图标。
- `color`: 更改文件上传的颜色。
- `variant`: 更改文件上传的变体。
- `size`: 更改文件上传的大小。

## 用法

```vue
<UFileUpload
  <!-- props here -->
/>
```
