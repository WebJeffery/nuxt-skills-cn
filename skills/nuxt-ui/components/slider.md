# 滑块

用于在范围内选择数值的输入。

> 基于 [Reka UI Slider](https://reka-ui.com/docs/components/slider)

## 关键属性

- `step`: 设置滑块的增量值。
- `orientation`: 更改滑块的方向。
- `color`: 更改滑块的颜色。
- `size`: 更改滑块的大小。
- `tooltip`: 在滑块拇指周围显示带有当前值的 [Tooltip](/docs/components/tooltip)。
- `disabled`: 禁用滑块。
- `inverted`: 视觉上反转滑块。

## 用法

```vue
<USlider
  <!-- props here -->
/>
```
