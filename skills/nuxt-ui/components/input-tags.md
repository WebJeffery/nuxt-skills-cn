# 标签输入框

显示交互式标签的输入元素。

> 基于 [Reka UI InputTags](https://reka-ui.com/docs/components/tags-input)

## 关键属性

- `placeholder`: 设置占位符文本。
- `color`: 更改标签输入框聚焦时的环颜色。
- `variant`: 更改标签输入框的外观。
- `size`: 调整标签输入框的大小。
- `icon`: 在标签输入框内显示 [Icon](/docs/components/icon)。
- `avatar`: 在标签输入框内显示 [Avatar](/docs/components/avatar)。
- `loading`: 在标签输入框上显示加载图标。
- `disabled`: 禁用标签输入框。

  <!-- props here -->
/>
