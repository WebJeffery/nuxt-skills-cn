# 进度条

显示任务进度的指示器。

> 基于 [Reka UI Progress](https://reka-ui.com/docs/components/progress)

## 关键属性

- `max`: 设置进度条的最大值。
- `status`: 在条形图上方显示当前进度值。
- `animation`: 将进度的动画更改为反向轮播、摆动条或弹性条。
- `orientation`: 更改进度的方向。
- `color`: 更改滑块的颜色。
- `size`: 更改滑块的大小。
- `inverted`: 视觉上反转进度条。

/>
