# 文本域

用于输入多行文本的文本域元素。

## 关键属性

- `rows`: 设置行数。
- `placeholder`: 设置占位符文本。
- `autoresize`: 启用文本域高度的自动调整。
- `maxrows`: 设置自动调整时的最大行数。
- `color`: 更改文本域聚焦时的环颜色。
- `variant`: 更改文本域的变体。
- `size`: 更改文本域的大小。
- `icon`: 在文本域内显示 [Icon](/docs/components/icon)。
- `avatar`: 在文本域内显示 [Avatar](/docs/components/avatar)。
- `loading`: 在文本域上显示加载图标。
