# 仪表板侧边栏

可调整大小和可折叠的侧边栏，用于在仪表板中显示。

## 关键属性

- `resizable`: 使侧边栏可调整大小。
- `collapsible`: 当在屏幕边缘附近拖动时，使侧边栏可折叠。
- `side`: 更改侧边栏的侧边。
- `mode`: 更改侧边栏菜单的模式。
- `toggle`: 自定义在移动设备上显示的 [DashboardSidebarToggle](/docs/components/dashboard-sidebar-toggle) 组件。
