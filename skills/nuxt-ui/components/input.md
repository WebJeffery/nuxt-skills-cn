# 输入框

用于输入文本的输入元素。

## 关键属性

- `type`: 更改输入类型。
- `placeholder`: 设置占位符文本。
- `color`: 更改输入框聚焦时的环颜色。
- `variant`: 更改输入框的变体。
- `size`: 更改输入框的大小。
- `icon`: 在输入框内显示 [Icon](/docs/components/icon)。
- `avatar`: 在输入框内显示 [Avatar](/docs/components/avatar)。
- `loading`: 在输入框上显示加载图标。
- `disabled`: 禁用输入框。

## 用法

```vue
<UInput
  <!-- props here -->
/>
```

## 插槽
