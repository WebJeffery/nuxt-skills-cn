# PIN 输入框

用于输入 PIN 码的输入元素。

> 基于 [Reka UI PinInput](https://reka-ui.com/docs/components/pin-input)

## 关键属性

- `type`: 更改输入类型。
- `mask`: 将输入视为密码。
- `otp`: 启用一次性密码功能。
- `length`: 更改输入框的数量。
- `placeholder`: 设置占位符文本。
- `color`: 更改 PIN 输入框聚焦时的环颜色。
- `variant`: 更改 PIN 输入框的变体。
- `size`: 更改 PIN 输入框的大小。
- `disabled`: 禁用 PIN 输入框。
