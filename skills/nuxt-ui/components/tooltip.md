# 工具提示

悬停在元素上时显示信息的弹出窗口。

> 基于 [Reka UI Tooltip](https://reka-ui.com/docs/components/tooltip)

## 关键属性

- `text`: 设置工具提示的内容。
- `kbds`: 在工具提示中渲染 [Kbd](/docs/components/kbd) 组件。
- `content`: 控制工具提示内容的渲染方式，例如其 `align` 或 `side`。
- `arrow`: 在工具提示上显示箭头。
- `disabled`: 禁用工具提示。

## 用法

```vue
<UTooltip
  <!-- props here -->
/>
```
