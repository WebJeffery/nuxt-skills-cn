# 时间输入框

用于选择时间的输入。

> 基于 [Reka UI InputTime](https://reka-ui.com/docs/components/time-field)

## 关键属性

- `color`: 更改时间输入框的颜色。
- `variant`: 更改时间输入框的变体。
- `size`: 更改时间输入框的大小。
- `icon`: 在时间输入框内显示 [Icon](/docs/components/icon)。
- `avatar`: 在时间输入框内显示 [Avatar](/docs/components/avatar)。
- `disabled`: 禁用时间输入框。

## 用法

```vue
<UInputTime
  <!-- props here -->
/>
```
