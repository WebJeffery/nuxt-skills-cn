# 按钮

可以用作链接或触发操作的按钮元素。

## 关键属性

- `label`: 设置按钮的标签。
- `color`: 更改按钮的颜色。
- `variant`: 更改按钮的变体。
- `size`: 更改按钮的大小。
- `icon`: 在按钮内显示 [Icon](/docs/components/icon)。
- `avatar`: 在按钮内显示 [Avatar](/docs/components/avatar)。
- `loading`: 显示加载图标并禁用按钮。
- `disabled`: 禁用按钮。
- `class`: 覆盖按钮的基本样式。
- `ui`: 覆盖按钮的插槽样式。

## 用法

```vue
  <!-- props here -->
/>
```
