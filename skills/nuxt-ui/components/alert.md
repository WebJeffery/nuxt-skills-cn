# 警告

用于引起用户注意的提示。

## 关键属性

- `title`: 设置警告的标题。
- `description`: 设置警告的描述。
- `icon`: 显示一个 [Icon](/docs/components/icon)。
- `avatar`: 显示一个 [Avatar](/docs/components/avatar)。
- `color`: 更改警告的颜色。
- `variant`: 更改警告的变体。
- `close`: 显示一个 [Button](/docs/components/button) 来关闭警告。
- `actions`: 向警告添加一些 [Button](/docs/components/button) 操作。
- `orientation`: 更改警告的方向。
- `class`: 覆盖警告的基本样式。
