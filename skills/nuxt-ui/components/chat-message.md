# 聊天消息

显示带有图标、头像和操作的聊天消息。

## 关键属性

- `parts`: 使用 AI SDK v5 格式显示消息内容。
- `side`: 在左侧或右侧显示消息。
- `variant`: 更改消息的样式。
- `icon`: 在消息旁边显示 [Icon](/docs/components/icon) 组件。
- `avatar`: 在消息旁边显示 [Avatar](/docs/components/avatar) 组件。
- `actions`: 在消息下方显示操作，当悬停在消息上时将显示这些操作。

## 用法

```vue
<UChatMessage
  <!-- props here -->
/>
```
