# 分隔符

水平或垂直分隔内容。

> 基于 [Reka UI Separator](https://reka-ui.com/docs/components/separator)

## 关键属性

- `orientation`: 更改分隔符的方向。
- `label`: 在分隔符中间显示标签。
- `icon`: 在分隔符中间显示图标。
- `avatar`: 在分隔符中间显示头像。
- `color`: 更改分隔符的颜色。
- `type`: 更改分隔符的类型。
- `size`: 更改分隔符的大小。

## 用法

```vue
<USeparator
  <!-- props here -->
/>
```
